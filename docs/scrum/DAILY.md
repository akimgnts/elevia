# Daily — Format Pilotage

Date : ___

## 3 Questions (30 secondes chacune)

### 1. Hier → Sprint Goal

> [Ce qui a avancé vers le Sprint Goal. Pas une liste de commits.]

### 2. Aujourd'hui → Sprint Goal

> [Ce qui va avancer vers le Sprint Goal. 1-2 actions max.]

### 3. Blocage ?

> [Oui/Non. Si oui : 1 phrase + qui peut débloquer.]

---

## Règles

- Pas de reporting détaillé
- Pas de discussion technique
- Si ça prend > 2 min → c'est une réunion, pas un daily
- Focus : le Sprint Goal avance-t-il ?
